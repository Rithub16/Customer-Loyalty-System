# Customer Loyalty Points System

## Project Overview

The Customer Loyalty Points System is a full-stack web application developed using React, Node.js, and Express.js. It allows users to create customers, calculate loyalty points based on purchase amounts, add bonus points, and view a summary of customer loyalty points.

---

## Features

* Create Customer
* View Customer List
* Add Purchase
* Calculate Loyalty Points
* Add Bonus Points
* Display Total Customers
* Display Total Loyalty Points

## Setup Instructions

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Backend Setup

```bash
cd backend
npm install
node src/index.js
```

Backend runs at:

```
http://localhost:5000
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

---

## Loyalty Points Logic

* Every ₹1 spent earns **1 loyalty point**.
* Bonus points can be added manually.
* Customer data is stored temporarily in memory using a JavaScript array.
